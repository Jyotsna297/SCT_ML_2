# SCT_ML_02 - Customer Segmentation using K-Means Clustering

## Objective
Group retail store customers based on their purchasing behavior using K-Means Clustering.

## Dataset
Mall Customer Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Steps Performed
1. Loaded the dataset
2. Selected Annual Income and Spending Score features
3. Used Elbow Method to find optimal clusters
4. Applied K-Means Clustering
5. Visualized customer segments

## Output
- Elbow Method Graph
- Customer Segmentation Graph
- Clustered Customer Dataset

## Conclusion
K-Means clustering successfully grouped customers into different segments based on spending patterns and annual income.
